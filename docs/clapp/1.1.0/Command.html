Command

new Command(options)

Source:
  • Command.js, line 7

A Command that can be bound to an App. A command represents a single function that achieves a single purpose. The command can receive arguments and flags, and can return a string that will be redirected to the app output (See: onReply).

Example
var foo = new Clapp.Command({
	name: "foo",
	desc: "does foo things",
	fn: function(argv, context) {
		console.log("foo was executed!");
	},
	args: [
		new Clapp.Argument({
			name: "testarg",
			desc: "A test argument",
			type: "string",
			required: false,
			default: "testarg isn't defined"
		})
	],
	flags: [
		new Clapp.Flag({
			name: "testflag",
			desc: "A test flag",
			alias: 't',
			type: "boolean",
			default: false
		})
	],
});

app.addCommand(foo);

Name Type Attributes Default Description
name string

The command's name. The command will be invoked by writing the app prefix followed by a space and the command's name.
fn function

The function that will be executed when the command is invoked. The function receives the following params: argv and context. For more information, see Defining-the-command-function.
desc string

A description of what the command does.
args Array.<Argument> <optional>

An array with every argument supported by the command. See Argument.
flags Array.<flag> <optional>

An array with every flag supported by the command. See flag.
